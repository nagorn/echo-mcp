# v0.2.0 - MCP Standard Agent Guidance

## Summary

Echo MCP v0.2.0 adds MCP-standard agent guidance surfaces to help AI coding
agents understand how to use Echo MCP and when manual mock behavior is not
contract-validated.

## Added

- MCP initialize instructions.
- Workflow-aware tool descriptions.
- Tool annotations.
- Guidance prompts.
- Guidance resources.
- Structured `configure_behavior` warnings, guidance, and
  `suggested_next_actions`.

## Compatibility

- Existing manual mock behavior remains supported.
- REST data-plane responses are unchanged by agent guidance.
- The new guidance fields are additive in MCP control-plane results.
- Strict MCP clients may need to tolerate additional structured output fields.
- Agent behavior may vary by MCP client, especially for prompts and resources.

## Not Included

- No `echo-mcp.yaml`.
- No full OpenAPI-first runtime.
- No provider-specific Stripe simulator.
- No public contract repository.
